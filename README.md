# Bank App

[![CircleCI](https://circleci.com/gh/umrashrf/genoa-bank.svg?style=svg)](https://circleci.com/gh/umrashrf/genoa-bank)

## Installation

```
$ pip install https://github.com/umrashrf/genoa-bank.git
```

## Setup Database

```
$ python -m bank.db
```

## Start Web Socket Server

```
$ python -m bank.api
```

## Tests

```
$ python -m unittest
```
